# Spears

This mods introduces the **spear** in versions before they were added (before 1.21.11)

> The spear has a **jab attack** and can also just be used as swords. **It can also be enchanted with Lunge I/II/III** which makes the jab attack boost the player forward while sacrifycing hunger points. The spear **deals more damage based on *velocity (player speed)*!**

**Adds:**
- Wooden/Stone/Iron/Copper/Gold/Diamond/Netherite spears
- Lunge I/II/III enchantments

## Made for:
- **NeoForge (main)**
- Forge
- Fabric
- _Quilt (beta)_
